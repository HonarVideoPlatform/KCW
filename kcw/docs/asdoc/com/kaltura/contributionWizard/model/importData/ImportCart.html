currentlyProcessedImportVoproperty
public var currentlyProcessedImportVo:BaseImportVO

The import item that is currently being proccessed. "Proccess" can be any kind of action that is being applied to any kind of BaseImportVO object. This is a marker variables to enable the view to show information of the proccessed file (for example, the currently uploaded file)
